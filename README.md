# Farsi Digits Dataset Generator



این پروژه یک ابزار قدرتمند و انعطاف‌پذیر برای تولید **داده‌های مصنوعی (Synthetic Data)** جهت آموزش مدل‌های هوش مصنوعی و پروژه‌های OCR است. با استفاده از این اسکریپت، می‌توانید هزاران تصویر متنوع از اعداد فارسی را به صورت خودکار تولید کنید.



## 🚀 ویژگی‌های کلیدی

- **تنوع فونت:** استفاده از فونت‌های مختلف فارسی موجود در پوشه مخصوص.

- **تغییرات تصادفی (Augmentation):** 

  - جابجایی پیکسل‌ها (Jitter) برای شبیه‌سازی دست‌خط و خطاهای چاپی.

  - چرخش تصادفی (از ۱۵- تا ۱۵+ درجه) برای افزایش دقت مدل.

  - تغییر سایز فونت به صورت داینامیک.

- **فرمت استاندارد:** خروجی تصاویر در ابعاد 32x32 پیکسل و به صورت سیاه و سفید (Grayscale).

- **سازماندهی خودکار:** دسته‌بندی تصاویر در پوشه‌های مجزا برای هر عدد.
